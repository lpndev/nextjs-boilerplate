## Getting Started

#### Clone repository and rename

```shell
git clone https://github.com/lpndev/template.git
mv template [project-name]
cd [project-name]
```

#### Initiate project

```shell
bun install
bun run dev
```

## Optional steps

#### Update dependencies

```shell
bun update
```

#### Check code

```shell
bun run format
bun run lint
```

#### Build locally

```shell
bun run build
bun run start
```
