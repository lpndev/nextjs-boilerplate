## Getting Started

#### Clone repository and rename

```shell
git clone https://github.com/lpndev/nextjs-boilerplate.git
mv nextjs-boilerplate [project-name]
cd [project-name]
```

#### First commit (optional)

```shell
git add . && git commit -m "initial commit"
```

#### Initiate project

```shell
bun install
bun run dev
```

## Optional steps

#### Update dependencies

```shell
bun update
```

#### Check code

```shell
bun run format
bun run lint
```

#### Build locally

```shell
bun run build
bun run start
```
